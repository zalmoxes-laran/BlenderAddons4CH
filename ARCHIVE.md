# Archive — defunct tools (historical record)

Tools that were once in this catalog but **no longer have a usable current home**
(dead sites, abandoned repos, discontinued products, or features absorbed into
Blender). They are recorded here for the historical record and removed from the
active catalog (`data/addons/`).

Each entry was re-verified online on **2026-06-09** (fan-out web research). Where
a modern replacement exists, it is noted — most live in the active catalog
already.

---

## Superseded by Blender's own evolution

The need is now met by Blender core or a different catalog entry.

- **Material Converter** — `material-converter` · materials — Converted Blender Internal (incl. Photoscan GLSL) materials to Cycles. *Blender Internal removed in 2.8; same-name repo unrelated.*
- **io_mesh_xyz** — `io-mesh-xyz` · point-cloud — Imported/exported XYZ position triplets to generate a point cloud. *Old 2.6 wiki gone; → use **Point Cloud I/O**.*
- **Import Bundler (.out)** — `import-bundler` · photogrammetry — Imported the `bundle.out` file from Bundler (SfM). *Standalone importer gone; → use the **Photogrammetry Importer**.*
- **Particle morphing tools** — `particle-morphing` · animation — Particle morphing. *`allblue.pl` download dead; deprecated.*
- **Texture Atlas** — `texture-atlas` · baking — Pack/bake multiple objects into a texture atlas (Esau/Geraskin/Barton). *Removed from Blender 2.8+; Google Code dead, only legacy mirrors.*
- **Blend Library** — `blend-library` · libraries — A linked-asset library. *Old 2.6 wiki page 404; → use the built-in **Asset Browser**.*
- **Blender Cloud textures** — `blender-cloud-textures` · libraries — Use Blender Cloud textures out of the box. *Add-on archived/unmaintained; → moved to **Blender Studio**.*
